Virtualbox

Install Virtualbox 4.3.4/4.2.20 or later from Virtualbox website - any older version are not tested and might break.
Choose a dedicated user to run Virtualbox & Hyperbox under, in this case : hyperbox. This is required to isolate and manage the environment under a single user.
sudo useradd -m hyperbox Remove WebSrv Authentication as it is not supported yet: sudo su - hyperbox -c "vboxmanage setproperty websrvauthlibrary null" Configure WebSvc
sudo nano /etc/default/virtualbox Recommanded config to add (hyperbox user must exist before)
VBOXWEB_USER=hyperbox Start WebSrv sudo /etc/init.d/vboxweb-service start

Java

Install : sudo apt-get install default-jre-headless Check as root that you get a successful version info: $ sudo java -version You should get something along these lines : java version "1.6.0_27"

Hyperbox

Download the latest Hyperbox wget http://hyperbox.altherian.org/download/hbox_server-latest-linux_amd64.run Run the installer
sudo sh hbox_server-latest-linux_amd64.run Start Hyperbox Daemon
sudo /etc/init.d/hboxd start Output and errors are logged to <INSTALL_DIR>/log/hboxd.log by default
